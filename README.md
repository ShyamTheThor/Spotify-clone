# 🎵 Spotify UI Clone (Frontend)

A frontend clone of the **Spotify web player interface**, built using **HTML and CSS**, focusing on layout design and visual accuracy.

🔗 **Live Demo:** https://spotify-clone-eight-jade.vercel.app

---

## 🚀 Features

* 🎧 Spotify-like UI layout
* 🧱 Sidebar navigation (Home, Search, Library)
* 🎵 Music player section (UI only)
* 🎨 Styled components using CSS
* 📱 Responsive design 

---

## 🛠️ Tech Stack

* HTML5
* CSS3
